Clonetris

About

Clonetris is the classic stack-em up... with a twist!

Whatever the block the player uses to complete a line of bricks,
that same block type must be used to clear the next line.

After the second line of bricks is cleared, the user is free to use any
brick that they choose to clear the next line.

That block type must then be used to clear the following line, etc...
